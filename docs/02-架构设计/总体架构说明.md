# 总体架构

```text
CloudLens Frontend
    ↓
CloudLens Backend
    ├─ Cloud Accounts
    ├─ Aliyun ECS/RDS ReadOnly API
    ├─ Huawei ECS/CES ReadOnly API
    ├─ Alert / AI / KB
    └─ File Upload Extension
    ↓
SQLite + Aliyun OpenAPI + Huawei Cloud SDK + Optional OSS
```

当前默认使用本地 SQLite：

- `data/cloud/cloud.db`
- `data/control/control.db`
- `data/kb/knowledge.db`

默认不依赖 OSS，也不依赖独立数据库。

补充说明：

- 当前前后端主线围绕阿里云 ECS/RDS 与华为云 ECS 监控展开
- 系统资源监控属于旧支线，不再作为当前产品主线继续推进
