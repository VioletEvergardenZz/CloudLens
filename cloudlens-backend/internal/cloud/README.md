# cloud 目录地图

`cloud/` 是云厂商适配器层。这里负责把阿里云、华为云等 SDK 调用包装成 CloudLens 内部更稳定的只读模型。

## 当前结构

| 目录 | 作用 |
| --- | --- |
| `aliyun/` | 阿里云 ECS、RDS、云监控只读查询 |
| `huawei/` | 华为云 ECS、RDS、CES 云监控只读查询 |
| `common/` | 云厂商间可复用的通用类型或工具 |

## 维护规则

- 适配器只做只读查询和字段转换，不做资源写操作。
- 云 API 局部失败时，尽量返回可用部分和错误详情，避免前端整页断流。
- 新增云厂商时，优先保持和现有 ECS/RDS 字段语义对齐，再考虑抽象复用。
- 真实凭据不得写入测试、文档或日志。
